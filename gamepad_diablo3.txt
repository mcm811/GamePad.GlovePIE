감동을 받아 계속 개인적으로 수정하고 개선(?)을 해온 글로프 파이 스크립트를 공개합니다.
(수정 하다보니 코드를 대부분 재작성 하게 되서 완전 다른 놈이 되어버렸습니다)

오랜동안 게임패드로 디아3를 하면서 느낀 장단점을 뽑자면,
장점: 키보드+마우스 조합 보단 케릭을 무빙을 좀더 세밀하게 할수 있다.
      소파에 편한 자세로 디아3를 할수 있다.
단점: 키보드+마우스 조합 보단 타겟팅 작업이 힘들다.
      편한 자세로 하다보면 더 오래 해서 더 피곤 해질 수가 있다.

적용 가능한 대균 단계(스탠기준)는 어느정도 되는가?
공부모드(신개폭): 대균 63단계 클리어
법성모드: 대균 59단계 클리어 (법성은 실력 중요한 역활인데, 제 실력이 별로라서^^)
황야바바(불카쌍수): 대균 솔플 53단계 클리어 (일균 시험석 작업용으로 키웠다가 적당히 파밍이 되서 테스트해본 결과)
독침모드: 대균 64단계 클리어 (이건 SSK클랜 시절, 클랜분들께 버스 받은거라 60단계 근처까지 가능하다 정도로 참고하시면 됩니다)
결론: 대균 40~60단계 정도 즐기기에는 크게 문제가 없다.

파일링크: ﻿https://drive.google.com/file/d/0B0P3gUU3gmRYX2kyUVhjNGxsMEU/view?usp=sharing﻿
실행방법: 파일을 받아 압축을 푸시고, gamepad_run.cmd 파일을 클릭해서 실행을 해주면 됩니다.
게임내 설정 사항: 디아블로3 단축키 설정에서 "모든창 닫기를 F9, 강제 이동을 F10"으로 설정해주세요.
(단축키를 다른 것으로 설정해서 사용하고 싶으신분은 텍스트 에디터로 gamepad_diablo3.pie을 열어서 Key.F9 와 Key.F10을 다른 키로 치환을 해주면 됩니다. 기타 추가적인 세부 설정(개폭 주기, 스킬 사거리 등등)들도 수정 가능합니다.)

게임내 사용방법:
// 모드 설정 커맨드:
// LB+LB: 법성 모드, LB+RB: 공부 모드, LB+LT: 황야바바 모드, LB+RT: 독침 모드
// 모드 해제 커맨드: 설정 상태에서 다시 LB 한번
// 각종 안내창 열기: LB+A(프로필), LB+B(순위표), LB+X(커뮤니티), LB+Y(친구)
// 자동 수리 / 분해: 수리창 열린 상태에서 LB+UP
// (대)균열 신청: LB+DOWN (가방 (7,4), (7,5), (8,5) 세칸의 석(파편,시험석,균열석)을 사용) - 좌상 기준 (0,0)칸 부터 시작

// 깃발: Down
// 마을: Right 길게
// 가방: Right 짧게
// 엔터: Up (확인창 수락용)
// 스킬/정복자스텟/추종자 스킬: Left 누르면 로테이션
// 지도: Start (패드 가운데 왼쪽 버튼)
// 메뉴: Back (패드 가운데 오른쪽 버튼)

// 1번 스킬: A
// 2번 스킬: X
// 3번 스킬: Y
// 4번 스킬: RB
// 기본 스킬 버튼은 일정시간 누르고 있으면 자동 발사 토글

// 마우스 왼쪽 스킬: RT
// 마우스 오른쪽 스킬: LT
// 케릭 이동: LS
// 마우스 이동: RS
// 물약: Left Thumb (왼쪽 스틱 클릭)
// 마우스 포인트 위치 초기화: Right Thumb (왼쪽 스틱 클릭)
// 모든창 닫기 및 아이템 루팅: B (누르고 케릭을 움직이면 아이템 루팅)

P.S. 막상 설명을 작성할려고 보니 "격하게 아무것도 안하고 싶다" 상태가 되어버려서,
     좀더 친절하고 상세히 설명을 드리지 못한점 죄송합니다.
